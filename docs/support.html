Safe to use

VoltPeek does not need internet access and does not upload your data. It only reads local battery and power info. Open source under MIT.

If macOS shows “VoltPeek.app” Not Opened, that is a signing/notarization warning — not a malware finding. Click Done (not Move to Trash), then allow the app in System Settings → Privacy & Security → Open Anyway, or right-click the app → Open.

Known limitations

  • Desktop Macs without a battery show limited or unavailable fields.
  • Adapter name, wattage, voltage, and current depend on what IOKit exposes for your charger.
  • Time-to-full and battery-runtime estimates appear only when macOS provides them. Runtime for a fully charged Mac becomes measurable after unplugging.
  • Very short refresh intervals (e.g. 0.5 s) use more CPU; prefer 2–3 s for everyday use.
  • Menu bar Battery / Watts / Both icons follow the system menu bar appearance (light or dark).
  • GitHub / Homebrew builds are not Apple-notarized yet; first launch may need Open Anyway.

Window and background behavior

The red close button closes only the main window; battery monitoring, the configured menu-bar item, and power-status alerts continue in the background. Use ⌘Q or Quit in the popover to exit VoltPeek completely.

Bug reports

Include: macOS version, Mac model (Apple Silicon / Intel), VoltPeek version (About), steps to reproduce, a screenshot if relevant, and optionally a report from Diagnostics → Copy Report.
